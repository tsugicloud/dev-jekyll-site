TSUGI-DEV

This is a free, hosted developer instance of Tsugi. It is reset daily at midnight Eastern. It runs the tip of the master branch of Tsugi. You can test applications on this site, learn how to administer a Tsugi instance, or use it for demonstrations.

Accessing this Server

For wide open dev servers the following are the defaults for access.

Tsugi admin: the mascot of the Sakai project - all lower case

Database account: The subdomain of the url. For example if the url was https://zap.tsugicloud.org/, the database user would be 'zap'

Database password: db_ followed by the admin password like db_happy

If these do not work, contact the creator of this site.
